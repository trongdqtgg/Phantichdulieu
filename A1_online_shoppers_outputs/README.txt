# Kết quả phân tích online_shoppers_intention.csv

Thư mục gồm:
- phan_tich_online_shoppers.py: code Python hoàn chỉnh để chạy lại.
- online_shoppers_intention_cleaned.csv: dữ liệu sau tiền xử lý.
- reports/*.csv: báo cáo phân loại biến, mô tả thống kê, outlier, so sánh trước/sau.
- charts/before_preprocessing/*.png: ảnh biểu đồ trước tiền xử lý.
- charts/after_preprocessing/*.png: ảnh biểu đồ sau tiền xử lý.

Cách chạy:
1. Đặt file online_shoppers_intention.csv cùng thư mục với file phan_tich_online_shoppers.py
2. Cài thư viện nếu cần:
   pip install pandas numpy matplotlib
3. Chạy:
   python phan_tich_online_shoppers.py
